# TypeScript Utility Library

A collection of 20 independent utility functions for common programming tasks.

## Structure
- Each function in its own file: `src/utils/{name}.ts`
- Each file exports one function
- No dependencies between functions
- Pure TypeScript, no external packages

## Functions to Implement

1. `slugify` - Convert string to URL slug
2. `capitalize` - Capitalize first letter of each word
3. `truncate` - Truncate string with ellipsis
4. `debounce` - Debounce function calls
5. `throttle` - Throttle function calls
6. `deepClone` - Deep clone an object
7. `flatten` - Flatten nested arrays
8. `unique` - Get unique values from array
9. `groupBy` - Group array items by key
10. `chunk` - Split array into chunks
11. `shuffle` - Randomly shuffle array
12. `pick` - Pick specific keys from object
13. `omit` - Omit specific keys from object
14. `merge` - Deep merge objects
15. `isEmpty` - Check if value is empty
16. `isEqual` - Deep equality check
17. `randomInt` - Generate random integer in range
18. `sleep` - Promise-based delay
19. `retry` - Retry async function with backoff
20. `memoize` - Memoize function results
